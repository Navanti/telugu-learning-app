# triage

## Purpose
Assess incoming roadmap items or issues and route them to the right agent with clear next steps.

## Workflow
1. Classify item type (bug, feature, refactor, chore).
2. Estimate impact, urgency, and dependency risk.
3. Map ownership to manager/frontend/nlp/learning-design/qa.
4. Split into implementation and validation tasks when needed.
5. Flag escalation cases and unresolved ambiguities.

## Outputs
- Priority/impact classification.
- Assigned owner agent and supporting agents.
- Initial task breakdown with escalation notes.
